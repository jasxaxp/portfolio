Dimensional Drift

This projects uses Object-Oriented Programming (OOP) and is a text adventure game. The player is teleported into a place where gates lead to different dimensions and worlds. The gates appear in different areas of the town. The player accidently is swept into one and must find the spell book in order to escape and go back home.

Space Invaders: Reworked

This game is Space Inavaders Reworked (my version), the aliens come in vertically, players can use the up, left, right, and down arrows to move anywhere they want. The player must take care of their lives, and health. To win the player must reach a certain level and score.


Visualization Of Covid Cases

This project focuses on the design, implementation and testing of Python programs to process data files to allow for visualization and analysis of covid cases between Korea and USA. In the line graph, viewers are able to analyze that Korea has significantly less deaths than the United States

Hangman

This project involves coding and testing of a program that facilitates playing the Hangman word game.Hangman is a popular word game. In this game, the player is given some number of blanks representing the name of a movie, an item or an actor and they have to guess the name using at most k number of chances.
