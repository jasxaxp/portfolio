Redesign K-Drama Webpage (Case Study)

This is a ux/ui design project for redesigning a wikipedia page.
If you have any questions about this project, feel free to contact me by emailing me or using the contact tab.


Back To UX/UI Design

Problem: The information page for Korean Dramas (K-dramas) located in wikipedia, is not user freindly and needs to be re-deisgned so users can understand and look for the content they like.

Solution (Goal): The solution is to create a website that can display the information about k-dramas in the most 'user-friendly' way by conducting user reserach to understand the needs of the user (to re-design the website).

Roles: UX/UI Designer, Prototyper and Web Designer

Tools: Adobe XD, Indesign

Skills: Wireframing, Prototyping

Time: Three to Four Weeks



Design Process:

Reserach ---> Content Study ---> User Personas ---> Information Architecture ---> Navigation (IA) ---> Wireframing ---> Finalize
