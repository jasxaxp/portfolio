Spell Well

This is wordpress and a ux/ui design project for a spelling game called "Spell Well".
If you have any questions about this project, feel free to contact me by emailing me or using the contact tab.

Back To Wordpress

Back To UX/UI Design

Client: This game helps those who are struggling with keeping up with school work and or are burnt out from studying and need for a new/ unique way of studying words. This idea uses a similar feature of “quizlet” where the users are able to input information and descriptions of what they need to memorize. In the site, users can input words that they need practice (or help with) along with their definitions, then during gameplay, players will try to guess words based on the definition that shows on screen.

Problem: This game needs a place to host the game to its users and where users can learn more about the game.

Solution (Goal): The solution is to create a website that can display the information about the game, a place for the game to run and a feedback area (where users can give feedback).

Roles: UI designer, Prototyper and Web Designer

Tools: Adobe XD, Dreamweaver, Wordpress

Skills: Wireframing, Prototyping, Usability Testing

Time: Four Weeks



Design Process:

Plan ---> Information Architecture ---> Visual Identity ---> Wireframing ---> Prototyping ---> Finalize
