Crystal Shop

This is a custom WordPress site for a startup business/organization. This website uses a custom WordPress website by using a standards-based front-end interface development framework (child-theme).This site represents the public digital face of a startup and features real content. Spell Well

This is a WordPress site for a spelling game called "Spell Well". This game helps those who are struggling with keeping up with school work and or are burnt out from studying and need for a new/ unique way of studying words. This idea uses a similar feature of “quizlet” where the users are able to input information and descriptions of what they need to memorize. In the site, users can input words that they need practice (or help with) along with their definitions, then during gameplay, players will try to guess words based on the definition that shows on screen.
