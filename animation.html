Animation Projects

These animation projects showcase the skills learnt using the principles of animation. If you have questions about these project, feel free to contact me by emailing me or using the contact tab.

Lip Sync

This animation uses the principles of lip sync. It lip syncs "when it rains, look for rainbows". This animation was done by creating a character's head with a library of lipsynch phonemes (mouth shapes).The mouth shapes that were created include for vowels, W, M, L, F, Consonant and Resting.

The Jellyfish

This animation uses the wave principles. This is a simple looping animation of a long string-like object waving. The object waves from a fixed point (from the jellyfish's face). To create this animation it was important to ensure I get the correct sense of motion, then use a guide as a guide to animate the object. The Animations demonstrates the principles of Follow through and Overlapping Action through timing and a sense of force on the object.


The Bouncing Ball

This animtion uses a simple circle/circular object and these three animations shows the following objects falling from a height, moving/bouncing across the screen and coming to rest: a heavy object (a bowling ball), a medium weight object (a tennis ball), and a rubbery elastic object (a rubber ball). These animations demonstrate timing/spacing, squash & stretch and a sense of weight.

The Walking Teddy Bear

This animation uses the princples of the walk cycle. This is a basic looping walk cycle, using the character I created. The character is a humanoid and it includes: Legs, Arms, A Head and A Torso that can bend at the Waist. The loop start on one foot (Foot A) then steps to the other foot (Foot B) and return to Foot A. This brings the loop back to the start.
